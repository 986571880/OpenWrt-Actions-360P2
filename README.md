# Actions-OpenWrt-Newifi3-d2

**基于 [Actions-OpenWrt](https://github.com/P3TERX/Actions-OpenWrt) 项目云编译的Openwrt固件**

该openwrt固件为Newifi3-d2专属

- Theme: 
- IP: 192.168.10.1
- Language: en; zh-cn



**编译插件:**

Firewall
Adguardhome
DNSfilter
ntpc
Passwall
samba4
smartdns
sqm
ttyd
UPNP
FTP文件服务器
FRPC
WOl网络唤醒
